# LIFT — PPL Workout Tracker

Premium iOS-optimized Push/Pull/Legs workout tracker. Each logged workout auto-syncs to GitHub.

## Features
- Push / Pull / Legs cycle with smart day detection
- Per-set weight + reps logging with checkoff
- Automatic PR tracking with e1RM calculation
- GitHub sync on every set logged
- Workout history + volume charts
- Streak tracking
- Native iOS feel (safe areas, momentum scroll, blur nav)

## Stack
- Vanilla HTML/CSS/JS — zero dependencies
- localStorage for offline-first data
- GitHub API for cloud sync
- Deployed via Vercel
